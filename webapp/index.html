Snake game

The goal of this game is to have the longest queue possible ! By eating bonus, your length will grow, by growing up, your score will as well. The longer you are, the more points you have. Be carefoul though, the game will be more and more complicated given the snake length.

This game is not available on this screen :( Consider opening this page in a web browser on your computer in order to enjoy using this game.

Commands

Q ← Move to the left D → Move to the right Z ↑ Move to the top S ↓ Move to the bottom P SPACE Pause/ Resume
